# Ingenier-a-de-software-y-Datos-PREICA2601B010059
Evidencia de Aprendizaje 1. Diseño Sistema de Información – Iteración 1


Sistema de Gestión - Autos Colombia
Este es un sistema de información desarrollado para la gestión de ingresos y salidas de vehículos en el parqueadero Autos Colombia.

Tecnologías utilizadas:
* **Backend:** Python con Flask.
* **Base de Datos:** SQLite con SQLAlchemy (ORM).
* **Frontend:** HTML5, CSS3 y Bootstrap 5.
